Lab 51: Pulling Shared Changes

Quick hop over to the clone repository and lets pull down the changes just pushed to the shared repo.

Execute:

cd ../cloned_hello
# (You should be in the cloned repository)
git remote add shared ../hello.git
git branch --track shared master
git pull shared master
