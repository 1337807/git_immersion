Lab 37: Review the Cloned Repository

Goals

  • Learn about branches on remote repositories.

Look at the cloned repository

Let’s take a look at the cloned repository.

Execute:

cd cloned_hello
git hist --all

Output:

$ cd cloned_hello/
$ git hist --all
* 9445ca0 - 2010-06-20 Updated Rakefile (HEAD, origin/master, origin/greet, origin/HEAD, master) [Jim Weirich]
* cb563df - 2010-06-20 Hello uses Greeter [Jim Weirich]
* 5b1ed69 - 2010-06-20 Added greeter class [Jim Weirich]
* 10f5258 - 2010-06-20 Added README [Jim Weirich]

Remote branches

This is almost the same as the original hello repository. However, not the list of branches on the “Updated Rakefile” line. You have a master branch, but you also have number of strangely named branches (origin/master, origin/greet and origin/HEAD). We’ll talk about them in a bit.
