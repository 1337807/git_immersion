Lab 2: More Setup

Goals

  • Get the tutorial materials setup and ready to run.

Steps

Get the Tutorial package.

Git the tutorial package from either:

  • The memory sticks provided in the tutorial
  • The URL http://onestepback.org/download/git_tutorial.zip

Unzip the tuturial

The tutorial package should have a main diretory “git_tutorial” with three sub-directories:

  • html — These html files. Point your browser to html/index.html
  • repos — Prepackages repos for later use
  • work — An empty working directory. Create your repos in here.
